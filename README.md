# 📝 To-Do List (Python CLI Application)

A simple command-line based To-Do List application built using Python.
This project allows users to manage daily tasks by adding, viewing, and deleting tasks.

---

## 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks with numbering
* ❌ Delete tasks by selecting task number
* 🔁 Continuous menu-driven interface
* 🚪 Exit confirmation for better user experience

---

## ▶️ How to Run the Project

### 1. Clone the repository

```
git clone https://github.com/YOUR_USERNAME/todo-list-python.git
```

### 2. Navigate to the folder

```
cd todo-list-python
```

### 3. Run the program

```
python todo.py
```
